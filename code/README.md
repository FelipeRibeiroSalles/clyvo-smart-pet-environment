# Código Fonte

Esta pasta contém o código-fonte principal do projeto Clyvo Smart Pet Environment.

## Arquivo Principal

- clyvo_pet.ino

## Funcionalidades Implementadas

- Leitura de temperatura com sensor TMP36
- Leitura de luminosidade com sensor LDR
- Acionamento automático de LED
- Alertas preventivos via Monitor Serial
- Monitoramento contínuo do ambiente

## Tecnologias Utilizadas

- Arduino UNO
- Linguagem C/C++
- Sensores analógicos
- Automação embarcada
